The Kana Theme

A warm, earthy dark theme inspired by the Gruvbox color palette. One consistent look across your editor, terminal, and website — wherever you write code.

Available for VS Code, Mac Terminal, Hugo, and Xcode.

The Palette

Kana's colors are drawn from the Gruvbox family — warm, muted tones that reduce eye strain without sacrificing contrast.

Variants

Every variant shares the same core palette, so switching between tools feels seamless.

VS Code

Install from the VS Code Marketplace and bring Kana's warmth to every file you edit.

Mac Terminal

A matching Terminal.app profile for macOS. Import once and your shell feels at home.

Hugo

The theme powering this very site. Add it to any Hugo project as a module or submodule.

Xcode

Extend the palette into Apple's IDE. Write Swift and Objective-C in the same warm tones.

Open Source

Every variant of the Kana theme is free and open source, licensed under MIT. Contributions and feedback are welcome on GitHub.
